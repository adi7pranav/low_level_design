| Dimension         | **Push**               | **Pull**            |
|-------------------|------------------------|---------------------|
| Who controls data | Subject                | Observer            |
| Coupling          | Lower                  | Higher              |
| Performance       | Usually better         | Can be worse        |
| API simplicity    | Cleaner                | More complex        |
| Testing           | Easier                 | Harder              |
| Best for          | Small frequent updates | Large complex state |
